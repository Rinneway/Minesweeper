# 💣 Minesweeper

Добро пожаловать в **Minesweeper** — классическую игру **«Сапёр»**, реализованную на **Python** с использованием **PyQt6**.

---

## 📦 Требования

Перед запуском проекта убедитесь, что у вас установлен:

- Python **3.13**
- Все необходимые зависимости из файла `requirements.txt`

Установить зависимости можно одной командой:

```bash
pip install -r requirements.txt
```

Или установить только PyQt6:

```bash
pip install PyQt6
```

---

## ▶️ Запуск проекта

После установки зависимостей выполните:

```bash
python minesweeper.py
```

---

## 🎮 Готовая версия игры

В папке **`dist`** находится готовая скомпилированная версия игры.

```
dist/
├── _internal/
└── minesweeper.exe
```

Для запуска достаточно открыть файл:

```text
dist/minesweeper.exe
```

> ⚠️ Не удаляйте папку **`_internal`**, так как она содержит необходимые библиотеки для работы программы.
