# 💣 Minesweeper

Добро пожаловать в **Minesweeper** — классическую игру **«Сапёр»**, реализованную на **Python** с использованием **PyQt6**.

---

## 📦 Требования

Перед запуском проекта убедитесь, что у вас установлен:

- Python **3.10+**
- Библиотека **PyQt6**

Установить PyQt6 можно командой:

```bash
pip install PyQt6
```

---

## ▶️ Запуск проекта

Если вы хотите запустить игру из исходного кода, выполните:

```bash
python minesweeper.py
```

---

## 🎮 Готовая версия игры

В папке **`dist`** находится готовая версия игры, которую можно запустить без Python.

```
dist/
├── _internal/
└── minesweeper.exe
```

Для запуска достаточно открыть файл:

```text
dist/minesweeper.exe
```

> ⚠️ Не удаляйте папку **`_internal`**, так как она содержит необходимые файлы для работы программы.
